Detailed Description

template<typename Resource, typename Identifier>
class ResourceHolder< Resource, Identifier >

This class is designed to hold game resources. game resources should only be loaded into memory once and used repeatedly. The RessourceHolder class is designed to facilitate this. The ResourceHolder class was not written by me and has been modified from the original code.
